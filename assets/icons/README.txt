GrowForge uses Unicode emoji and text-based icons throughout the UI.
To add custom icons, place PNG files (32x32 or 64x64) in this folder
and reference them with CTkImage in the UI code.

Suggested custom icons:
- logo.png (app logo)
- plant.png
- leaf.png
- water.png
- sun.png
- scissors.png
- flask.png
- bug.png
- chart.png
